Terminal Powered Seismology
===========================

# Requirements

  * [Rust and Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html)

# Installation

    cargo install --path .

# Usage

    termology <filename.mseed>

Shortcuts:

| Key        | Action           |
|:----------:|------------------|
| q          | quit             |
| j          | Scroll down      |
| k          | Scroll up        |
| +          | Show more traces |
| -          | Show fewer traces|
